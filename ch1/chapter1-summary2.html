1.1.2 Testing vs Debugging

Key Distinction: Testing finds problems, Debugging solves problems. They are complementary but distinct activities performed by different roles with different skills.
Real-Life Example: Login System Issue
Testing Scenario: "Tester tries logging in with correct password, but system rejects login and shows error message. Documents this defect with detailed steps."
Debugging Scenario: "Developer uses debugger to trace code, discovers case-sensitive comparison error. Fixes code and tests the fix."
