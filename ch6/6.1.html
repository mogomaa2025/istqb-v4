6.1 Tool Support for Testing

دعم الأدوات للاختبار - ISTQB Foundation Level v4.0

هدف التعلم: FL-6.1.1 (K2) Explain how different types of test tools support testing

الغرض من أدوات الاختبار

"Test tools support and facilitate many test activities."

أدوات الاختبار هي برامج وتطبيقات تساعد المختبرين وفرق التطوير على أداء أنشطة الاختبار بشكل أكثر كفاءة وفعالية. يمكنها أتمتة المهام المتكررة، وتوفير معلومات دقيقة، وتمكين أنواع من الاختبارات كان من الصعب أو المستحيل إجراؤها يدويًا.

ملاحظة هامة: "Any other tool that assists in testing (e.g., a spreadsheet is a test tool in the context of testing)". حتى جدول البيانات مثل Excel يمكن اعتباره أداة اختبار إذا تم استخدامه لإدارة حالات الاختبار.

فئات أدوات الاختبار

"Examples include, but are not limited to:"

Management tools

تسهل إدارة متطلبات الاختبار والعيوب والتكوين.

Static testing tools

تدعم المراجعات والتحليل الإستاتيكي للكود.

Test design and implementation tools

تسهل إنشاء حالات وبيانات الاختبار.

Test execution and coverage tools

تسهل التنفيذ الآلي للاختبارات وقياس التغطية.

Non-functional testing tools

تسمح بإجراء اختبارات غير وظيفية (مثل الأداء والحمل) التي يصعب تنفيذها يدويًا.

DevOps tools

تدعم مسار التسليم المستمر (CI/CD)، تتبع سير العمل، والبناء الآلي.

أمثلة على أدوات حقيقية

لربط الفئات النظرية بالواقع، إليك جدول يوضح أمثلة لأدوات شائعة في السوق لكل فئة مذكورة في المنهج.

Management tools

تزيد كفاءة عملية الاختبار عبر تسهيل إدارة دورة حياة تطوير البرمجيات، المتطلبات، الاختبارات، العيوب، والتكوين.

Examples: Jira, Azure DevOps, TestRail
Static testing tools

تدعم المختبر في أداء المراجعات والتحليل الإستاتيكي.

Examples: SonarQube, ESLint
Test design and implementation tools

تسهل إنشاء حالات الاختبار، بيانات الاختبار، وإجراءات الاختبار.

Examples: Postman (for API tests), Selenium IDE (for recording)
Test execution and coverage tools

تسهل التنفيذ الآلي للاختبارات وقياس التغطية.

Examples: Selenium WebDriver, JUnit, JaCoCo (for coverage)
Non-functional testing tools

تسمح للمختبر بأداء اختبارات غير وظيفية يصعب أو يستحيل أداؤها يدويًا.

Examples: JMeter, Gatling (for performance), OWASP ZAP (for security)
DevOps tools

تدعم مسار تسليم DevOps، تتبع سير العمل، عمليات البناء الآلي، والتكامل/التسليم المستمر (CI/CD).

Examples: Jenkins, GitLab CI, Docker, Kubernetes
Collaboration tools

تسهل التواصل.

Examples: Slack, Microsoft Teams, Confluence
